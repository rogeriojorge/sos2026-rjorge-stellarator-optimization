# Boundary Modes

## Prompt

Increase one non-axisymmetric boundary-mode amplitude in the toy scan and predict how QS residual, aspect ratio, and iota might change.

## Expected qualitative answer

The QS residual usually worsens first; some shape freedom can improve one metric while hurting another.

## Research extension

Replace the cached metric with an output from a real code path, then compare whether the design ranking changes.
