# Data

`data/vmec_equilibria/` stores small public VMEC equilibrium files fetched by `scripts/fetch_equilibria.py`.

`data/inputs/` stores small public input decks such as the SIMSOPT QA target.

`data/cached/` stores intentionally small educational fallback arrays generated by `scripts/generate_all_figures.py`.

`data/generated/` is for rerunnable local outputs and is ignored by git except for this directory placeholder.

Do not put prompt-package PDFs, PPTX files, zips, or screenshots here.
